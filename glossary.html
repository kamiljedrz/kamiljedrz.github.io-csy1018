Glossary

  • 1. Archive

This is a copy of any previous software, documents or files. These files are kept for further references.

  • 2. Accessibility

Accessibility allows the users with some health problems like hearing, or impaired sight to use your website by adding additional settings to the website.

  • 3. Application

A computer software that has been designed to help us perform a specific actions or processes.

  • 4. Ad banner

Add banners are used for online advertisements on a web page. These adverts link the user to their other websites.

  • 5. Alignment

Alignment is making sure that the content of the webpage is positioned in the way I have planned.

  • 6. Backup

This is a copy of any previous software, documents or files. These files are kept in case you damage or lose your original ones.

  • 7. Backlinks

Backlinks are also called pingbacks and trackbacks on blog websites. They are simply the links from the third-party websites, usually used as a reference.

  • 8. Blog

Short name for "web log". People write blogs and share their own posts about their hobbies, personal experiences, life stories etc.

  • 9. Bandwidth

Bandwidth is a speed rate the data is transferred, or can get transferred to a web host.

  • 10. Cache

Cache is used to store the information of the websites you have previously visited. Once the user used the same website multiple times, the website will load faster as the cache is able to restore the web content.

  • 11. Client Side

This refers to scripts that are running and being processed on the user’s device. This allows the website to load faster as all the processes doesn’t have to be processed on the server.

  • 12. Comment

When writing any HTML, CSS or JavaScript, if you comment the code, you probably know that it useful as you don’t have to read the whole code to know which code have the specific function or role.

  • 13. Compression

Compressions are used to decrease the file size of the files. By compressing them, you will reduce the quality and size not needed that will make your file take less storage. Most of the websites compress their images so they load faster on the website.

  • 14. Data

This is a raw facts and figures collected. They are used to form the complete information.

  • 15. Database

A database is a storage for any type of records.

  • 16. Domain

A domain refers to the address of the website. For example; www.facebook.com. Facebook.com is an overall domain address. All domains have their own unique IP address which is transferred to the server once we type the domain name into our browser.

  • 17. Encryption

It is a data that is scrambled using a software with a key to make sure no one understands data when viewed.

  • 18. Field

Field is a single item of data.

  • 19. File server

This is a computer that is connected to the network that keep data or files that can be accessed by other users connected to the same network.

  • 20. FTP

File Transfer Protocol is mainly used when we upload or transfer any files into the internet.

  • 21. Fork

This is a term for using someone’s work. In most cases, the work copied is improved and developed.

  • 22. GUI

Graphics user interface is usually an output device such as keyboard or mouse. This allows the user to interact with the web application.

  • 23. Gigabyte

Gigabyte is a storage capacity measure of the device. One gigabyte represents 1024 megabytes.

  • 24. HTML

Hypertext Mark-up Language is a set of instructions that tells the web browser how to load the code to display the web page.

  • 25. Input

Input is simply a data we input into the computer system.

  • 27. Information

Information is formed when we process data and put it into the structure that is understandable.

  • 28. Interface

Interface takes place when two systems equipment are connected to each other.

  • 29. Import

Import is used when the text, images, or other information’s are created by one program, and then transferred into the other program.

  • 30. JavaScript

It is a scripting language that is used to add functions, as well as improve the overall look of the website.

  • 31. JPEG

It is an image file format most widely used. It allows you to use lossy and lossless compressions without losing a lot of image quality.

  • 32. Kilobyte

Kilobyte is a storage capacity measure of the device. One kilobyte represents 1024 bytes.

  • 33. Login

Login is a process that identify the person on a computer using the password, email or username.

  • 34. Link

Link let the user navigate the user around web-pages by simply clicking on the desired text, image or product.

  • 35. Network

Network is created when a list of computers and computer systems are connected to each other to share data across them.

  • 36. Output

Output is the result of processing all the data and obtaining the result.

  • 37. Open Source

Open source is a code of a program that is publicly shared to be publicly used by others. This method is cost effective for many companies.

  • 38. Password

A security check that the username or email belong the correct person.

  • 39. Protocol

This is a set of rules that needs to go through before the communication take place.

  • 40. Private network

A network that is used to let the organisations communicate privately, rather than making the communication public that could be accessed by people from the outside of the company.

  • 41. PNG

It is an image file format that is often used for web based images. It does the job well on large area of the same colours or patterns. This format allows the images to get a transparent background.

  • 42. Resolution

It’s a large number of pixels that can be shown on certain devices screen. Most of the devices display resolution is 1920x1080.

  • 43. Registrar

This is a company where you purchase your domain name. Example of the website is NameCheap.

  • 44. Storage

This is when data is recorded or kept and saved if needed for further use.

  • 45. Template

A template is a file that is used to keep the same design for a whole website.

  • 46. URL

Uniform Resource Locator is the easy to write and remember by the user. It transfers you to the specific web address of your choice.

  • 47. Web browser

Web browser is a software application that process the codes which allows the user to view the images, texts or other things the Website can store.

  • 48. Web page

Webpage is usually a text document that is written with HTML and CSS.

  • 49. Wireframe

It is a design of the website that give you the visual view of the design and functions your website is going to have before creation.

  • 50. WWW

World Wide Web is a subnet that needs HTTP or HTTPS protocol that sends web page documents through the whole internet.
